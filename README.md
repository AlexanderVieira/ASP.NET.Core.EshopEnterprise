# eShopEnterprise
Projeto E-commerce desenvolvido em dotnet core 3.1 com C#. 

Arquitetura Micro Distribuida.

Tecnologias:
- Identity;
- JWT;
- EFCore;
- RabbitMQ;
- EasyNetQ;
- CQRS;
- Event Storage;
- Docker;
- Refit; 
- Retry com Polly;
- Fluent Validation;
- Fluent API;
- Razor Extensions;
- WebAPI Gateway BFF;
- SOLID;
- Dapper;
- BackgroundService;
- IAggregateRoot;
- UnitOfWork;
- ExceptionMiddleware;
- ViewComponets;
- ValidationProblemDetails (RFC7807 - especificação para tratameneto de mensagens de erro em APIs);
