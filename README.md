# e-shop-enterprise
Projeto E-commerce desenvolvido em dotnet core 3.1 com C#. 

Arquitetura Micro Distribuida.

Tecnologias:
- Identity;
- JWT;
- EFCore;
- RabbitMQ;
- EasyNetQ;
- CQRS;
- Event Storage;
- Docker;
- Refit;
- Retrofit;
- Polly;
- Fluent Validation;
- Fluent API;
- Razor Extensions;
- WebAPI Getway BFF;
- SOLID;
